# EzBulkNetDriveMapping

Ce petit logiciel est en grande partie faite pour mon usage personnel.

Il permets de connecter des lecteurs réseaux en masse.

## Fonctionnalités

- Charger un fichier csv (format : lettre;chemin)
- Ajouter à la voler des duos lettre/chemin après un chargement de fichier
- Ajouter des duos lettre/chemin sans chargement de fichier
- Supprimer tout les lecteurs réseaux déjà mappés.


## Attention toutefois
Aucune erreur ne sera indiqué pendant le mappage. Il faudra vous rendre dans le fichier de log qui se trouve
dans le même répertoire que l'éxécutable afin de voir si tout s'est bien passé.
